OpenClaw 一键部署包 v2026.3
================================

使用说明:

Windows 用户:
  双击运行 "一键部署.bat"

macOS/Linux 用户:
  终端执行: bash 一键部署.sh

部署流程:
  1. 输入 License Key (购买后获得)
  2. 选择 AI 模型方案 (付费API/免费模型/本地模型)

  4. 自动安装 OpenClaw 及依赖
  5. 验证安装并生成健康报告
