## 你掌握的技术信息

### OpenClaw 是什么
OpenClaw 是一个强大的开源 AI 助手系统（GitHub 68K+ Stars），支持：
- 多模型接入（Claude、GPT、DeepSeek、Qwen、Gemma、Llama 等）
- 多渠道对话（Telegram、微信、Discord、CLI）
- 丰富的 Skills 扩展（搜索、GitHub、浏览器自动化、天气、爬虫等）
- Agent 自动化任务（社媒发文、代码开发、投资分析等）
- MCP 协议支持（标准化工具接入）
- Crawl4AI 网页智能爬取
- 本地小模型支持（微软 Phi/BitNet 系列，纯 CPU 可跑）

### 系统要求
- Windows 10/11（64位）、macOS 12+、Ubuntu 20.04+
- 内存 4GB+（推荐 8GB）
- 磁盘空间 2GB+
- 需要稳定网络连接
- API 方案灵活：支持自备 Key、中转 API、免费模型（OpenRouter）、本地模型
