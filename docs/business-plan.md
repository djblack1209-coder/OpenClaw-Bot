## 四、防退款机制设计

### License 系统
1. **设备绑定**：License Key 首次激活时绑定设备指纹（MAC地址+CPU ID）
2. **激活次数限制**：每个 Key 最多激活 2 次（允许换电脑）
3. **时效性**：永久授权，但退款后自动失效
4. **远程验证**：每次启动时联网验证 License 状态

### 退款检测
1. 闲鱼订单状态监控（通过 Cookie 轮询）
2. 检测到退款 → 自动将 License 标记为 `revoked`
3. 用户下次启动时提示"License 已失效，请联系客服"
4. 记录退款用户黑名单（设备指纹），拒绝再次购买

### 技术实现
- License 服务器：复用现有 `deploy_server.py`
- 验证接口：`/api/license/verify`
- 吊销接口：`/api/license/revoke`（闲鱼监控触发）

## 五、自动发货链路

### 方案A：百度网盘自动发货（推荐）
**流程**：
1. 买家付款 → 闲鱼 WebSocket 收到订单事件
2. 自动生成 License Key（UUID + 签名）
3. 从百度网盘获取分享链接（预先上传好安装包）
4. 通过闲鱼消息发送：
   ```
   感谢购买！您的 License Key：XXXX-XXXX-XXXX
   
   下载链接：https://pan.baidu.com/s/xxxxx
   提取码：1234
   
   安装步骤：
   1. 下载解压
   2. 双击"一键部署"
   3. 输入上方 License Key
   4. 按提示操作即可
   
   遇到问题请联系客服，7天内免费技术支持！
   ```
5. 同时发送 Telegram 通知给卖家
6. 同时发送邮件备份（可选）

**优点**：
- 成本低（百度网盘免费）
- 用户熟悉度高
- 支持大文件（安装包可能 100MB+）

**缺点**：
- 百度网盘限速（可引导用户开会员）
- 分享链接可能失效（需定期检查）

---

### 方案B：自建下载服务器
**流程**：
1. 买家付款 → 生成专属下载链接
2. 链接格式：`https://你的域名/download?key=LICENSE_KEY`
3. 下载时验证 License，防止盗链
4. 下载完成后链接失效（或限制下载次数）

**优点**：
- 完全可控
- 下载速度快
- 可统计下载数据

**缺点**：
- 需要服务器和带宽（成本 ¥50/月起）
- 需要备案域名

**结论**：初期用方案A，月销量 >50 后考虑方案B
