    3.

    4. 
                  
  4. auth.json 里只需要 OPENAI_API_KEY，值填本站用户 Key；不要填上游供应商 Key。

    7. 
                  
  7. 写入通用配置后重启 Codex；同一会话先让模型记住一句话，再切换模型追问这句话，检查上下文是否自然衔接。
