## 内容骨架模板

### 小红书 - 干货帖模板

```
[标题：疑问句/否定句/数字句，12-20字]

[钩子段：1-2句话，直击痛点或抛出反常识观点]

[正文：3-5个要点，每个要点2-3句话]
要点1：xxx
要点2：xxx
要点3：xxx

[收尾：总结一句话 + 互动引导]

#标签1 #标签2 #标签3（3-5个高相关标签）
```

### 小红书 - 教程帖模板

```
[标题：场景+结果，让人知道看完能得到什么]

[前置说明：1句话说清楚这个教程解决什么问题]

Step 1：xxx
Step 2：xxx
Step 3：xxx
（配图标注关键步骤）

[效果展示：before/after对比]

[互动：你们还想看什么教程？评论区告诉我]

#标签
```

```

1/ [展开论述第一点]

```

```
[可选：附上相关链接或截图]
```

## 选题评分标准

每个选题发布前打分（满分10分，≥7分才发）：

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 痛点强度 | 30% | 目标受众是否真的有这个痛点？ |
| 标题吸引力 | 25% | 看到标题会不会想点进来？ |
| 实用价值 | 20% | 看完能不能立刻用起来？ |
| 话题性 | 15% | 会不会引发讨论/转发？ |
| 与OpenClaw关联度 | 10% | 能不能自然带到产品？ |

## 热点结合策略

### 可蹭热点类型

- AI行业大新闻（新模型发布、大厂动态）
- 考试季（考研/考公/高考/期末）
- 消费节点（双11/618/年货节 → AI比价）
- 健身季（春天减肥/夏天露肉）
- 职场节点（金三银四/年终总结/述职）
- 科技圈争议（AI伦理/裁员/行业变化）

### 热点响应速度

- S级热点（AI行业重大事件）：2小时内出内容
- A级热点（相关领域热门话题）：当天出内容
- B级热点（可结合的泛热点）：1-2天内出内容

## 发布节奏

### 小红书

- 周一至周五：每天1条（晚8-9点发布）
- 周六日：可选发1条生活向内容（上午10-11点）
- 内容比例：干货3 : 教程1 : 生活1 : 热点评论1（每周6条循环）

### X

- 每天1-2条（不固定时间，更随性）
- 重大热点随时发
- 周末可以少发或不发
